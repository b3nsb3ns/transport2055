## West Coast Express

### Integration with FVX

In this plan, the service is rerouted west of Port Coquitlam such that it can merge with the Fraser Valley Express dedicated passenger tracks, so that it can be run all-day rather than only for commuters. This avoids the Cascades Subdivision in Vancouver/Burnaby/Tri-Cities, which is busy with freight, and allow relatively, all-day service.

### Coquitlam Central Station

As for Coquitlam Central Station, it will be served with the [Tri-Cities Express](tcx), using the same passenger tracks as the West Coast Express.
