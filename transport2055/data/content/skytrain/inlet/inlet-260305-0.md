## Burrard Inlet SkyTrain Line + Hastings SkyTrain Line

### Inlet SkyTrain Line
This line is to address three things: rail access for the North Shore, deal with the congestion on the Second Narrows \(Ironworkers Memorial\) Bridge, and provide a rapid north-south corridor in Burnaby. 

### Hastings SkyTrain Line
The Hastings line is bundled with the Inlet line in this plan, as it provides a transfer that allows the Inlet line to have its maximum benefit, with those crossing the second narrows being given an easy way downtown \(and vice-versa\).
